# Seeds

Thư mục này chứa dữ liệu demo.

Seed cần có trong MVP:

1. Tài khoản demo: QTV, CBDT, GV, KTV, SV.
2. Ba phòng in-scope: `2B11`, `2B21`, `2B31`.
3. Ca học chính: tiết `1-4` và `7-10`.
4. Học phần/lớp học phần/tổ thực hành từ ảnh UIS.
5. Lịch thực hành confirmed từ scope.
6. Dữ liệu test ràng buộc:
   - trùng phòng,
   - trùng giảng viên,
   - phòng thiếu máy,
   - thiếu phần mềm,
   - ngày nghỉ,
   - phòng bị block.

Không đưa dữ liệu phòng ngoài scope vào demo chính.
